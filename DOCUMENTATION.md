# Документація Todo List Додатку

## Опис
Це React додаток для управління списком завдань (todos) з можливістю фільтрації, пошуку та перегляду деталей кожного завдання.

## Основні функції
1. Відображення списку завдань
2. Фільтрація за статусом (всі/активні/завершені)
3. Пошук за назвою завдання
4. Перегляд деталей завдання в модальному вікні
5. Відображення інформації про користувача для кожного завдання

## Структура проекту
```
src/
├── components/
│   ├── Loader/          # Компонент завантаження
│   ├── TodoFilter/      # Компонент фільтрації та пошуку
│   ├── TodoList/        # Компонент списку завдань
│   └── TodoModal/       # Компонент модального вікна
├── types/               # TypeScript типи
├── api.ts              # API функції
└── App.tsx             # Головний компонент
```
